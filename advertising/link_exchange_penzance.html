Link Exchange with Purely Penzance

 

Purely Penzance are looking to set up link exchanges with web sites that cater to the people and businesses of Penzance, Penwith & Cornwall. We offer highly contextual links due to the wide range of subjects covered on this website, which is great for SEO.

We can provide links from over 300 pages across. Links will be placed in the sponsored spot, underneath the main navigation to the right of each page, or at the bottom of each page where the link exchange text is placed.

If you would like to set up a link exchange, please contact us, including the web address you would like to be linked to plus any text you would like included. If you fit into the recently launched Arts section, you can get a full profile page and extra goodies for free. Get in touch for more info.

Static images & banners are accepted. These should be no bigger than 150x100.

 

Link exchange code

To link to us, please add the following (or similar) to your site, then send us message.

Purely Penzance - The guide to restaurants, entertainment, accommodation, shopping & commerce in Penzance, Cornwall
